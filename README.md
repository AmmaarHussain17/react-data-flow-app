# React Data Flow Application

## Overview
This project demonstrates API data fetching, caching, search filtering, and dynamic flow visualization using React.

## Features
- Fetch data from JSONPlaceholder API
- Cache data using localStorage
- Search & filter posts with debounce
- Dynamic node creation using React Flow
- Connect nodes with edges
- API data used inside flow nodes

## Tech Stack
- ReactJS
- Vite
- React Flow
- JavaScript

## How to Run
```bash
npm install
npm run dev
```

## Concepts Covered
- useEffect & useState
- API handling
- Caching
- Debouncing
- State management
- Dynamic UI rendering
